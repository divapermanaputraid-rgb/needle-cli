# FungiCode Product Requirements Document (PRD)

## Vision
FungiCode is an open-source, multi-provider AI coding CLI inspired by modern agentic coding assistants and terminal-first developer workflows.

## Target Audience
- Open-source contributors
- Indie developers
- Personal power users
- Developers who want a multi-provider terminal coding agent

## Core Features
1. Multi-provider support (OpenAI, Gemini, DeepSeek, 9Router)
2. Agentic looping workflow (plan, code, review)
3. Direct terminal integration (shell execution, git review)
4. Granular permission system (ask, auto-low-risk, yolo)
